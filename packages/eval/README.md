# @durion/eval

Durion evaluation capture and metrics.

This package provides tools for re-running captured inputs against new variants (models, prompts, configs) and recording outputs to track regression and quality over time.

## Installation

```bash
npm install @durion/eval
```

## Requirements

- Node.js >= 20
- PostgreSQL database
- Peer dependency: `@durion/sdk`

Please refer to the main repository documentation for setup and workflows.
